# План: Фильтрация записей на главном экране по цветовым тегам

## Цель

Добавить фильтрацию по цветовым тегам на главном экране через диалоговое окно. Набор цветов для выбора формируется только из реально существующих `colorTag` в записях. Фильтр должен работать совместно с поиском и сохраняться между перезапусками приложения.

## Контекст и ограничения

- Архитектура: MVVM + Clean Architecture, offline-only.
- Безопасность: не использовать `!!`.
- Логи: на русском.
- Кнопка фильтра находится в `TopAppBar` справа от сортировки.
- Иконка кнопки: `Palette` с визуальным состоянием:
  - без активного фильтра: `Icons.Outlined.Palette`;
  - при активном фильтре: `Icons.Filled.Palette`.
- UI фильтра: диалог с заголовком `Фильтр по цвету`, сетка цветов, кнопки `Применить` и `Сбросить`.
- Визуал выбора цвета в сетке: бордюр по аналогии с экраном создания/редактирования (`CreateEditScreen`).
